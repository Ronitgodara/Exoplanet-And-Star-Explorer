# Exoplanet & Star Explorer

An interactive astronomy dashboard built with Python, Streamlit, Plotly and MySQL.

## Features

* Planet Explorer
* Star Explorer
* Planet Type Distribution
* Discovery Trends Analysis
* Habitability Analysis
* Star Classification (O, B, A, F, G, K, M)
* NASA-Style Planetary System Visualization
* Search and Filter Functionality
* Interactive Plotly Charts

## Technologies Used

* Python
* Streamlit
* Plotly
* Pandas
* NumPy
* MySQL

## Dataset

The project contains data for 100+ exoplanets and 70+ stars stored in a MySQL database.

## Run Locally

```bash
streamlit run exoplanets.py
```
